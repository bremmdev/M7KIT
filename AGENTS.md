This is a React component library with a strict focus on accessibility and
design-system consistency.

## Accessibility rules (non-negotiable)

- All components MUST meet WCAG 2.2 AA requirements at minimum.
- Semantic HTML MUST be used before ARIA.
- ARIA roles, states, and properties MUST follow WAI-ARIA Authoring Practices.
- Keyboard navigation MUST be supported for all interactive components.
- Focus management MUST be explicit and testable.
- Color contrast MUST meet WCAG requirements.
- Components MUST be screen-reader compatible (NVDA, VoiceOver).

## Accessibility resources

- WCAG 2.2: https://www.w3.org/TR/WCAG22/
- WAI-ARIA Authoring Practices: https://www.w3.org/WAI/ARIA/apg/patterns/
- MDN Accessibility Guides: https://developer.mozilla.org/en-US/docs/Web/Accessibility
